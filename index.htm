Built with Rust

Mabor introduces a unique architecture based on tensor operation streams, fully optimized at runtime and auto-tuned for your hardware by a Just-in-Time compiler. This approach relies on Rust's ownership rules to precisely track tensor usage, a feature unattainable without a robust type system. It enables you to create highly dynamic models with the performance of an extensively optimized static graph.


Mabor adheres to the best practices of the Rust programming language, renowned for its reliability, enhancing development speed and helping you achieve your goals more quickly.
